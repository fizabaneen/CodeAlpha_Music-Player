# 🎵 Music Player App

A simple and responsive Music Player built using HTML, CSS, and JavaScript. This application allows users to play, pause, and control music playback with an easy-to-use interface.

## 🚀 Features

- ▶️ Play Music
- ⏸️ Pause Music
- ⏭️ Next Track
- ⏮️ Previous Track
- 🎚️ Progress Bar
- 🔊 Volume Control
- 📱 Responsive Design
- 🎨 Clean and Modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 How to Use

1. Open the application.
2. Select a song.
3. Click the Play button to start music.
4. Use Next and Previous buttons to switch tracks.
5. Adjust volume and progress using the controls.
